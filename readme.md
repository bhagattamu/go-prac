# Go Tools

### Run command

```sh
$ go run welcome.go
```

### Build command

```sh
$ go build welcome.go
```
It build go file to .exe in windows. Then run 

```sh
$ ./welcome
```

## Tools

```sh
$ go help
```

- Go has a built-in test suite. `Test` can measure performance of your code
- `Get` is for installing third-party packages
- `fmt` format your code
